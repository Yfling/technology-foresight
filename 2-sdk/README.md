## ajax接口数据格式规范

```javascript
    {
        status: 1, // 接口返回值状态码
        errMsg: '失败信息', // 接口返回值错误信息，正确则为空
        result: { // 所有的返回值都封装在result对象中
            code: '123456'
        }
    }
```
